# LinkedIn Post Draft

I have completed my first game in Java: Snake.

I built the project step by step with Java Swing, starting from an empty window
and a single moving square. During development I implemented:

- keyboard controls with arrow keys and WASD;
- snake growth and random food placement;
- wall and self-collision detection;
- persistent scoring and gradual speed increases;
- pause and restart mechanics;
- custom snake and fruit sprites;
- sound effects and audio controls;
- start and game-over screens.

This project helped me better understand Swing rendering, keyboard events,
collections, coordinate-based movement, timers, resource loading, audio, and
basic game-loop logic.

Source code: https://github.com/gerion88/snake-game-java

Next, I plan to continue improving my Java skills and build more small projects.
