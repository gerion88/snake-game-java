# Portfolio Description

## Short Version

Classic Snake game built from scratch in Java using Swing. The project includes
keyboard controls, textured snake rendering, random fruit, collision detection,
persistent scoring, increasing difficulty, audio controls, pause and restart
mechanics, and a custom game background.

## Extended Version

Snake Game is a learning project created to practise Java fundamentals and
desktop GUI development with Swing. The game uses a timer-driven update loop,
keyboard event handling, a list of coordinates for the snake body, collision
detection, random food placement, and custom rendering.

The project was developed incrementally: first displaying a window and a single
moving square, then introducing controls, food, growth, collisions, scoring,
restart and pause mechanics, progressive difficulty, persistent high scores,
sound effects, textured sprites, and polished game screens.

## Skills Demonstrated

- Java syntax and object-oriented programming
- Swing rendering and event handling
- Collections and coordinate-based movement
- Game-loop logic with `Timer`
- Collision detection
- Resource and audio loading
- Persistent settings with `Preferences`
- Incremental development and debugging
- Git and GitHub project preparation
